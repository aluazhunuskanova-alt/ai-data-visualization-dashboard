# AI Data Visualization Dashboard

## Overview
This project is a web-based data analysis dashboard built with FastAPI and Plotly.

## Features
- Upload CSV/Excel datasets
- Automatic data cleaning
- Interactive visualizations:
  - Scatter plot
  - Histogram
  - Bar chart
  - Box plot
  - Correlation heatmap
- Machine Learning prediction (Random Forest)
- Insight generation

## Technologies
- Python (Pandas, NumPy)
- FastAPI
- Plotly
- Scikit-learn
- HTML/CSS

## How to Run
```bash
uvicorn main:app --reload
